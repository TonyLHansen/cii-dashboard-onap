The CII questions are collected into these sections:
  • Analysis
  • Basics
  • Change Control
  • Quality
  • Security
  • Reporting
In addition, the CII questions have been categorized into these types:
  • Application Quality — The quality of the application itself.
  • FLOSS — FLOSS encouragement
  • Infrastructure — The quality of the infrastructure used to build and support the application.
  • People — Questions about the people building the application.
  • Project Quality — The quality of the project oversight.
ONAP-met refers to those CII questions for which we have an ONAP Project-wide answer that everyone can copy and paste. These responses can all be found on our ONAP CII wiki page. Almost ALL responses in this category should display the button .
